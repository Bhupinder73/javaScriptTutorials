Core Concepts of JavaScript

  • Arithmetic Operators - +, - , /, *, %
  • Increment/Decrement Operators - ++, --
  • Comparison Operators.
  • Variable - containers used to store data.
  • Ways to Create Variables - var(global scoped), let(block scoped), const(value not change)
  • DataTypes - Primitive(number, string, boolean, null, undefined) & non-primitive(Array, Object, Function)
  • null vs undefined
  • typeof
  • if-else(ladder, nested)
  • Logical Operators (&&, ||, !)
  • Functions
  • Loop - Code runs multiple times based on a condition.
  • Loop vs Function - Funtion(reusable block of code) & Loop(repeated execution of code)
  • Callbacks - is a function passed as an argument to another function, which is then invoked inside the outer function.
  • Anonymous Function - function without name. Used as arguments to other functions or assigned to a variable.
